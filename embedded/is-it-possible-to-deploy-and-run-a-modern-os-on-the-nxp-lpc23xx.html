Is it possible to deploy and run a modern OS on the NXP LPC23xx?

up vote 3 down vote favorite

I bought a MCB2300 evaluation board in 2010. I'm wondering if it is possible to run an OS like Windows 10 RT or an embedded Linux on it.

Euripedes already said it. Lets see why:

My first computer back in the 90's had Windows 95 on it, which is ridiculously small and resource friendly compared to modern OSes. My computer back then had a Pentium processor with a 100 MHz clock, 16MB of RAM and a hard drive with 800MB, where the OS required about 200-ish MB.

Your LPC23xx has a max frequency of 72MHz, 64KB of RAM and 512KB of Flash. It even comes without the ability to use an external Flash (correct me if I'm wrong). So the answer is nope, nope and nope.

The LPC23xx family are ARM7TDMI microcontrollers. You won't be able to run an OS like Linux on it for any practical application.

You can run an OS like FreeRTOS or run a bare metal application.

As comments below it's possible to run ucLinux on ARM7TDMI but you'll need external RAM and possibly an external memory to store the OS file systems.

A ANSI/POSIX RTOS option is NuttX.
