How to utilize SIMD instructions at higher level C programming?

up vote 2 down vote favorite
1

we are using Intel processors (quad core server platform) which are 64 bits CPUs. The OS is Linux and the compiler is gcc.

We need a lot of computations (specifically addition and subtractions) of 8 bit numbers. We want to achieve eight of 8-bit signed subtractions in 1 64 bit instructions in one go.

So naturally we can program it using SIMD instructions that provide efficient executions. One of the sources I was referring was to use vec_add as of here. This specific work was referring to altivac. I am sure, there exist the intel x86 equivalent way of doing it.

My question is, since this is quite a common requirement now in multimedia and networking related functions, are there any C programming level techniques (basically without explicit assembly language) to use vector additions and vector subtraction SIMD instructions?

simd 64-bit-computing assembly-language optimization

Use the Intel Performance Primitives.

This library is highly optimized to do exactly what you need. You must use glibc or eglibc (uclibc will not work).
