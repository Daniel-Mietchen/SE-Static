How to use Android kernel with Linux filesystem?

up vote 5 down vote favorite

The Qualcomm apq8084 SoC I am working on has only partial Linux support in the Linux mainline or in Linaro. It looks like the only option I have is to use the Codeaurora Android msm-3.10 git tree that does have full support for the sound and camera devices that I use. There seems to be no other kernel tree with complete support for this SoC.

My device will be an embedded Linux device using a Buildroot filesystem, so my question is, when I build a kernel from a Codeaurora kernel tree that is intended for Android, do I need to to make any configuration changes in order for the kernel to work properly with a Linux root filesystem?
